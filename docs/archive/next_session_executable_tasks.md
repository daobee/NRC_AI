### Task P0-1: 采集伤害基准测试集

**目标**: 构建 20-30 组游戏内真实伤害数据，用于校准公式

**输入**:

**输出**: 
- `data/damage_benchmark_cases.json`
- 包含每个用例的精灵/技能/环境参数及实测伤害

**执行步骤**:

1. **制定采样策略** (0.5 天)
   - [ ] 选定 5-10 个战斗场景
   - [ ] 每个场景设计 3-4 个伤害数据点
   - [ ] 涵盖：克制/被克制/无效/能力修改/天气

2. **采集数据** (1.5 天)
   - [ ] 进入游戏战斗场景
   - [ ] 截图每次伤害
   - [ ] 记录完整参数：
     ```json

       "attacker": "精灵名",

       "target": "目标精灵",

       "target_stats": {"hp": ..., "def": ...},

     ```

3. **验证数据完整性** (0.5 天)
   - [ ] 检查所有必填字段
   - [ ] 检查参数值范围（正常吗？）
   - [ ] 去重与清理

**成功标准**:
- [x] 20+ 个数据点采集完成
- [x] 覆盖至少 5 种属性克制情况
